# 🚀 Daily Python Challenge

Welcome to my Python journey! This repository is a living portfolio of my daily progress in mastering software development with Python.


## 📖 AI Usage (Important)
Small but important note about AI usage in this repo: I am using AI to boost my learning and productivity. Not a single line of code in those projects will be written by AI, however documentation, READMEs and other things might be. AI is substituting for a teacher, QA assistance and senior developer for me. It is explaining topics to me and then I instruct it to listen to me explain topics back to it in my own language, so it can check if I understand them correctly. 

The only way I use AI's code is through Intellisense and ONLY if I understand it before running the code itself. If I can't properly understand what code is doing before I run it - I copy it for later, rewrite it myself, and then use agent to review differences.

In short I wrote this to ensure you: every single concept and skill that you will or will not see in this repository is my own. Including **smart** AI usage.

## 🎯 The Goal
I am building one standalone project every single day. This challenge is not just about writing syntax; it's about consistency, architectural thinking, and tackling increasingly complex problems—from basic algorithms and state management to Object-Oriented Programming (OOP) and beyond.

## 📂 Repository Structure
Each folder in this repository represents a new day and a new, fully functional project.
* Every project is completely self-contained with its own `main.py` entry point.
* Inside each folder, you will find a dedicated `README.md` explaining the project's business logic, the technical requirements, and the key engineering concepts I learned or applied that day.
